# DSA Taller 8 - Railway MIAD

Proyecto de despliegue de API usando Railway y CircleCI.

## Descripción
API de predicción de abandono de clientes bancarios (Bank Churn).

## Tecnologías
- FastAPI
- Railway (PaaS)
- CircleCI (CI/CD)
- Python 3.10

## URL de la API
https://bankchurn-api-production-a33a.up.railway.app

## Endpoints
- `/docs` - Documentación interactiva
- `/api/v1/health` - Health check
- `/api/v1/predict` - Predicción de churn
